# 见微 PicMark

见微 PicMark 是一款轻量、本地、离线可用的图片查看与标注工具。它保留了普通看图软件的直接体验，又补上了常用的编辑能力：标注、裁切、旋转、加水印、格式转换和项目保存。

它适合不想打开大型修图软件、也不想把图片上传到网页工具里的用户。打开本地图片后，可以直接查看、编辑、保存，必要时还能保存为 `.picmark` 项目文件，之后继续修改。

## 适合谁

- 需要快速给图片加框、箭头、文字、水印的人
- 经常处理证件、票据、资料图、截图存档和说明图的人
- 想要本地离线处理图片，不想上传到在线工具的人
- 觉得专业修图软件太重、普通看图软件又不能编辑的人
- 需要在低配置电脑、低分辨率屏幕上编辑图片的人
- 不想为了基础标注、水印、裁切这类常用功能反复遇到会员限制的人

## 核心能力

- **看图更轻**：支持 JPG、PNG、BMP、WebP 等常见图片格式。
- **标注够用**：框选、圈选、箭头、文字、画笔、马赛克、裁切都在同一个工具里完成。
- **水印完整**：支持文字平铺、证件网格、单个文字、图片 Logo 和批量水印。
- **项目可继续**：保存为 `.picmark` 后，可以保留图层和编辑状态，下次继续改。
- **格式转换**：把当前图片转换为常见格式，保留预览和输出选择。
- **轻量更新**：可自动检查新版本，但不会强制弹窗、自动下载或自动安装。
- **查看/编辑分离**：查看时界面清爽，编辑时进入深色工作台，状态更明确。
- **右键与打开方式**：安装后可加入 Windows 打开方式、右键菜单和 `.picmark` 文件关联。

## 为什么选择它

PicMark 不追求把所有专业功能都塞进去，而是把普通人最常用的图片处理流程做短、做直观。

- 相比大型修图软件，它更轻，启动和操作成本更低。
- 相比在线设计工具，它不需要账号、不需要会员，也不需要上传图片。
- 相比把基础功能拆成会员项的软件，它把常用标注、水印、裁切和格式转换放在本地免费流程里。
- 相比普通图片查看器，它不止能看图，还能标注、裁切、加水印和保存项目。
- 相比截图工具，它更适合反复编辑已有图片，不会因为退出截图流程而丢掉进度。

## 低配置与旧系统友好

PicMark 面向普通办公电脑设计，重点照顾低配置设备和低分辨率屏幕：

- 支持 Windows 7 SP1、Windows 10 和更新版本。
- 基于 .NET Framework 4.7.2，适合旧系统部署。
- 避免沉重后台任务，不做全盘扫描和大规模缩略图预生成。
- 关键面板和弹窗按低分辨率屏幕使用场景做过压缩和滚动适配。
- 功能优先本地完成，核心流程不依赖网络服务。

## 隐私与更新

PicMark 默认在本地处理图片，不上传图片、文件名、文件路径或编辑内容。

- 自动检查更新：启动后延迟进行，最多每 3 天一次，可关闭。
- 匿名使用统计：需要用户允许后才会启用，可随时关闭。
- 在线率统计：不做实时在线监控，只建议用匿名日活、周活、月活估算。

详细说明见 [隐私说明](docs/PRIVACY.md)。

## 安装

从 [Releases](https://github.com/Tsang12140/picmark/releases) 下载：

- `PicMark-Setup-0.2.3.exe`：推荐普通用户使用，可选注册右键菜单、打开方式和 `.picmark` 工程文件。
- `PicMark-portable-0.2.3.zip`：免安装版本，解压后直接运行 `PicMark.exe`。

## 从源码构建

要求：

- Windows 7 SP1 或更高版本
- .NET Framework 4.7.2
- Visual Studio 2022 或 MSBuild

构建免安装版：

```powershell
.\build-portable.ps1
```

构建安装包需要安装 Inno Setup 6，然后运行：

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" .\installer\PicMark.iss
```

安装包会输出到：

```text
installer\Output
```
