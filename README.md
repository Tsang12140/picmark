# 见微 PicMark

见微 PicMark 是一个本地优先、离线可用的轻量图片查看与标注工具。它面向很具体的日常场景：打开一张已经存在于电脑里的图片，查看、标注、裁切、加水印、保存项目，并在之后继续修改。

它不是截图工具，也不是复杂的设计软件。PicMark 更接近一个带编辑能力的本地图片查看器：打开快、界面直观、不上传图片、不要求账号，也不会在用户明确导出或覆盖前破坏原图。

## 适合做什么

- 给车位图、证件照、票据、资料图、流程图做本地标注
- 给图片添加文字、箭头、框选、圈选、马赛克、水印和 Logo
- 保存为 `.picmark` 项目文件，之后继续编辑
- 在查看器和编辑器之间快速切换
- 对图片做裁切、旋转、翻转、格式转换和保存
- 批量给同一文件夹里的图片添加水印

## 为什么做 PicMark

很多标注需求并不是“截一张图，马上发出去”，而是围绕一张本地图片反复修改。截图工具在这种场景里容易中断工作流：截图状态会占住屏幕，误退出后标注可能丢失，多次截图也会损失画质。

大型设计软件又太重，按钮太多，学习成本高。在线设计工具还会遇到上传图片、账号会员、网络环境和隐私顾虑。

PicMark 的目标是保留图片查看器的直接和轻巧，同时补上最常用的标注、项目保存、水印和导出能力。

## 核心功能

- **本地图片查看**：支持 JPG、PNG、BMP、WebP 等常见格式。
- **轻量标注**：支持框选、圈选、箭头、文字、画笔、马赛克和裁切。
- **智能手绘美化**：手绘圆、矩形、箭头等形状后，可自动美化为更规整但仍保留手写感的图形。
- **水印工具**：支持证件网格、文字平铺、单个文字、图片 Logo 和批量水印。
- **项目文件**：可保存为 `.picmark`，保留图层与编辑状态，之后继续修改。
- **格式转换**：支持常见图片格式之间的转换，并提供预览与输出目录选择。
- **查看器模式**：浅色、克制，适合日常看图。
- **编辑器模式**：深色、专注，适合标注和处理图片。
- **打开方式集成**：安装后可出现在 Windows 的打开方式、右键菜单和 `.picmark` 工程文件关联里。

## 产品优势

- 相比复杂修图软件，PicMark 只保留高频标注流程，普通用户更容易上手。
- 相比在线设计工具，PicMark 本地运行，不需要上传图片，也不依赖会员。
- 相比普通图片查看器，PicMark 不止能看图，还能直接标注、裁切、水印和保存项目。
- 相比截图工具，PicMark 更适合持续编辑同一张本地图片，不会因为退出截图流程而丢失工作进度。
- 相比专业图像软件，PicMark 更轻，适合普通办公电脑和低配置机器。

## 设计原则

- **本地优先**：核心功能不依赖网络服务。
- **保护原图**：默认不改写原始图片，除非用户明确选择覆盖。
- **少即是快**：常用操作优先，低频功能收进菜单或弹窗。
- **低配置友好**：避免沉重的后台扫描、大量缩略图生成和昂贵的实时计算。
- **给普通人用**：按钮、文案和流程尽量直接，不要求用户理解专业图像概念。

## 安装与使用

从 [Releases](https://github.com/Tsang12140/picmark/releases) 下载：

- `PicMark-Setup-0.2.0.exe`：推荐给普通用户，支持安装、右键菜单、打开方式和 `.picmark` 工程文件关联。
- `PicMark-portable-0.2.0.zip`：免安装版本，解压后直接运行 `PicMark.exe`。

## 从源码构建

要求：

- Windows 7 SP1 或更高版本
- .NET Framework 4.7.2
- Visual Studio 2022 或 MSBuild

构建免安装版：

```powershell
.\build-portable.ps1
```

构建安装包需要安装 Inno Setup 6，然后运行：

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" .\installer\PicMark.iss
```

安装包会输出到：

```text
installer\Output
```
